# GetNugetPackageStats

A Node.js package that gets a nuget package statistics

## Usage

```js
npm install nuget-getstats
```

Then,

```js
var nugetStats = require('./index');
nugetStats.GetNugetPackageStats('printclassinstance').then(function(data) {
    console.log(data);
})
```
